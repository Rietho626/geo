# Geography Quiz App

A simple app for creating geography quizzes.

Quiz modes are either multiple-choice or type-in.
Quiz Topics are country-flags/capitals/inhabitants/area

Other featrues include:
  - choice of continents to draw countries from
  - time limit per question
  - question types (eg. country -> capital, capital -> country or mixed questions)
  - theme selection
  - language selection (german/english)

The app was created using vanilla JS and HTML/CSS, focussing on state handling and DOM manipulation w/o a Framework/Library.
The use of AI was limited to technical inquiries, as a result no code used was directly generated.
